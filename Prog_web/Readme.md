# 🔴 Programação Web

> Construindo a base do desenvolvimento para a Web.

---

> [!ABSTRACT]
> Repositório acadêmico da disciplina **Programação Web** da FATEC.
>
> Durante a disciplina foram desenvolvidos exemplos práticos utilizando HTML, CSS, JavaScript e PHP, explorando desde a criação de páginas simples até aplicações com interação dinâmica.
>
> O objetivo foi compreender os fundamentos do desenvolvimento web e a construção de interfaces para usuários.

---
